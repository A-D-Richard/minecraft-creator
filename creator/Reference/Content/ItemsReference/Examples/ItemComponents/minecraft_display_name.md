# minecraft:display_name

`minecraft:display_name` sets the item's display name within Minecraft:Bedrock Edition.

## Parameters

|Name |Default Value  |Type  |Description  |
|:----------|:----------|:----------|:----------|
|value|*not set* |String | Set the display name for an item.|

## Example

```json
"minecraft:display_name":{
    "value": "secret_weapon"
}
```
