# minecraft:dye_powder

`minecraft:dye_powder` sets the color for an item that is treated as dye.

> [!NOTE]
> There are 16 kinds of dye.

## Parameters

|Name |Default Value  |Type  |Description  |
|:----------|:----------|:----------|:----------|
|color|*not set* |String |color value to use |

## Example

```json
"minecraft:dye_powder":{
    "color" : "purple"
}
```
