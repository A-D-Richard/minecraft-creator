# minecraft:unit_cube

`minecraft:unit_cube` is a `String` component that specifies that a unite cube is to be used with tessellation.

## Default Parameter

|Default Value|Type |
|:----|:----|
|*not set*| String|

## Extra Parameters

`minecraft:unit_cube` does not use any extra parameters.

## Example

```json
"minecraft:unit_cube":
```
