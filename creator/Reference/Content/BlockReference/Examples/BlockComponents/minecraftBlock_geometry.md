# minecraft:geometry

`minecraft:geometry` is a `string` component that sets the geometry model to use for the block.

## Default Parameter

|Default Value|Type |
|:----|:----|
|*not set*| String|

## Extra Parameters

`minecraft:geometry` does not use any extra parameters.

## Example

```json
"minecraft:geometry": "cube"
```
