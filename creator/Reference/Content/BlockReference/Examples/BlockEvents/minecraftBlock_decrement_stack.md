# decrement_stack

`decrement_stack` is an Event Response that will reduce an item stack.

## Extra Parameters

`decrement_stack` does not use any extra parameters.

## Example

```json
"decrement_stack":{
    "origin": [-8.0, 0.0, -8.0],
    "size": [16.0,16.0,16.0],
}
```
