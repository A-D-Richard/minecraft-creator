# swing

`swing` is an Event Response that will cause the actor to swing.

## Extra Parameters

`swing` does not use any extra parameters.

## Example

```json
"swing":{
    "origin": [-8.0, 0.0, -8.0],
    "size": [16.0,16.0,16.0],
}
```
