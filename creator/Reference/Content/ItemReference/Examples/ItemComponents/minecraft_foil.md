# Item Documentation - minecraft:foil

`minecraft:foil` sets whether the item has a shimmery glint to its appearance.

## Example

```json
"minecraft:foil": false
```

## Vanilla entities examples

### appleEnchanted

```json
"minecraft:foil": true
```
