# minecraft:behavior.silverfish_merge_with_stone

`minecraft:behavior.silverfish_merge_with_stone` allows an entity to enter a stone block.

> [!NOTE]
> This behavior can only be used by `silverfish` entity types.

## Example

```json
"minecraft:behavior.silverfish_merge_with_stone":{
    "priority": 2,
}
```
