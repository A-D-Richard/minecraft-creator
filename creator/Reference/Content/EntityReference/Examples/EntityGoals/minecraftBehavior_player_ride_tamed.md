# minecraft:behavior.player_ride_tamed

`minecraft:behavior.player_ride_tamed` allows an entity to be rideable after being tamed by a player.

## Parameters

`minecraft:behavior.player_ride_tamed` does not use any parameters.

## Example

```json
"minecraft:behavior.player_ride_tamed":{
}
```
