# minecraft:behavior.squid_dive

`minecraft:behavior.squid_dive` allows an entity to dive underwater.

> [!NOTE]
> This behavior can only be used by the `squid` entity type.

## Example

```json
"minecraft:behavior.squid_dive":{
    "priority": 2,
}
```
