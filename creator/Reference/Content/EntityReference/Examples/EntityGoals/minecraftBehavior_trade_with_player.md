# Entity Documentation - minecraft:behavior.trade_with_player

`minecraft:behavior.trade_with_player` allows an entity to focus on a player that will initiate a trade.

## Example

```json
"minecraft:behavior.trade_with_player":{
    "priority": 1
}
```
