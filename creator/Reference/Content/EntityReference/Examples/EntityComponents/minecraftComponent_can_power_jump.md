# Entity Documentation -  minecraft:can_power_jump

`minecraft:can_power_jump` allows an entity to power jump similar to the `horse` entity.

## Example

```json
"minecraft:can_power_jump":{
}
```

## Vanilla entities examples

### donkey

```json
"minecraft:can_power_jump": {
}
```
